# Liv & Elle Learning Arcade (Phase 1 Scaffold)

This repo is the foundation for the Elite Adaptive Learning System for Liv (7, Grade 2) and Elle (5, Grade 1).

## Required env vars (set in Vercel)
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- OPENAI_API_KEY
- PARENT_PIN

## Local dev
```bash
npm install
npm run dev
```
